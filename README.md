## Azure Virtual Machine 생성 방법 

본 repository는 Azure VM을 생성하는 방법을 정리

PowerShell을 이용하여 VM 생성

 | 파일    | Linux VM 생성 방법 |
 | -------- | -----------------  |
 | Windows 2016 datacenter VM 생성 |`powershell/create_winvm.ps1`      | 
 | Linux Ubuntu VM 생성     | `powershell/create_linuxvm.ps1`              |
